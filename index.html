# Philosophy

> Predictable

What you write is what runs. No hidden behavior, no runtime surprises.

> Type-Safe

If it compiles, it works. Rust's guarantees, extended to web development.

> Fast

Zero-cost abstractions. Your app is as fast as hand-written code.

use elif::{Application, Get, Response};

#[tokio::main]
async fn main() {
    let app = Application::new()
        .route("/", Get(hello));
    
    app.run("0.0.0.0:3000").await;
}

async fn hello() -> Response {
    "Hello, Rust!".into()
}
use elif::{Application, Router, Json};
use serde::{Serialize, Deserialize};

#[derive(Serialize, Deserialize)]
struct User {
    id: u64,
    name: String,
}

async fn get_users() -> Json<Vec<User>> {
    let users = fetch_from_db().await;
    Json(users)
}
use elif::{middleware::Auth, Session};

async fn protected_route(
    session: Session,
    user: AuthUser,
) -> Response {
    format!("Welcome, {}!", user.name).into()
}

app.route("/dashboard", Get(protected_route))
   .middleware(Auth::required());
use elif_orm::{Model, Query};

#[derive(Model)]
struct Post {
    id: i64,
    title: String,
    content: String,
}

let posts = Post::query()
    .filter(|p| p.published.eq(true))
    .order_by(|p| p.created_at.desc())
    .fetch_all().await?;

# Built-in batteries

[■]

Async/Await Native

First-class async support. No callbacks, no promises.

[▲]

Type-Safe ORM

SQL queries checked at compile time. Migrations included.

[●]

WebSocket Support

Real-time communication with backpressure handling.

[★]

Authentication

Session management, JWT, OAuth2. Secure by default.

[◆]

Background Jobs

Queue system with retries and scheduled tasks.

[✓]

Testing Suite

Integration tests, mocks, and fixtures built-in.
